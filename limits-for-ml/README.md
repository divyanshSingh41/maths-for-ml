# Importance of LIMITS in Machine Learning

Limits are fundamental to:

* Understanding derivatives

* Computing gradients numerically

* Analyzing loss functions

* Learning how Gradient Descent works

Without limits, concepts like backpropagation and optimization cannot be understood properly.
